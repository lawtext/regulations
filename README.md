# 中国法律法规在线文库

## 说明

文档数据（`content`目录中）来自：[:link: 国家法律法规数据库](https://flk.npc.gov.cn/search)，
使用`markitdown`将`docx`文件转化为`markdown`格式，并进行适当格式化处理，文本内容仅供参考，如有问题，可以提交 issue。

目前包括全国性法律法规（超过两千件，包括已废止的文档），而地方性法规数量较多暂未纳入网站展示。
`docx/doc`文档参见仓库（定期自动更新）：
